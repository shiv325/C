# Bubble Sort in C

    void bubbleSort (int arr[], int size) {
        int temp;
        for (int i = 0; i < size - 1; i++) {
            for (int j = 0; j < size - i - 1; j++) {
                if (arr[j] > arr[j + 1]) {
                    temp = arr[j + 1];
                    arr[j + 1] = arr[j];
                    arr[j] = temp;
                }
            }
        }
    }

Parameters :
  - Integer Array
  - Size

Return :
  - Return Type -> None
